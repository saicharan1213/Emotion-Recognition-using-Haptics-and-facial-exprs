* To produce the output of the project, please execute final_code.py, can use "python final_code.py" in command line to produce output
* face epression dataset link - https://www.kaggle.com/datasets/jonathanoheix/face-expression-recognition-dataset
* Emotion_detection_cnn_model.h5 file has the CNN model weights stored.
* Touch_recog_model_ann.pkl has the ann model weights stored.
* History_keras.pkl contains the loss and accuracy information of the CNN  model.
* Scratch_clean.csv, smooth_clean.csv, and tap_clean.csv are the haptic data for different types of touch.
* target_haptics.csv contains some samples of haptics data which is used to run the final_code.py file.
